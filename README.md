MediaVerify is a full-stack web application designed to combat digital misinformation by providing a simple and powerful tool for detecting fake media. The system uses a dual-engine AI approach to analyze and classify both textual news articles and images, helping users distinguish between authentic and manipulated content.

### Features

-   **Multi-Modal Analysis:** A comprehensive solution that can analyze both text and images within a single platform.
-   **Text-Based Fake News Detection:** Utilizes a Natural Language Processing (NLP) model to classify news articles, social media posts, and other text as likely 'real' or 'fake'.
-   **Image Forgery Detection:** Employs a deep learning Computer Vision model to detect manipulated images, including AI-generated deepfakes.
-   **Simple Web Interface:** An intuitive, easy-to-use frontend that allows users to simply paste text or upload an image to get an instant analysis.

---

### Technology Stack

-   **Backend:** Python, Flask
-   **Machine Learning:** TensorFlow/Keras, Scikit-learn, Pandas, OpenCV
-   **Frontend:** HTML, CSS, JavaScript
